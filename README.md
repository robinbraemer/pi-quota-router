# pi-quota-router

Quota-aware multi-account routing for normal Pi.

The extension will keep the selected Codex model fixed while choosing among equivalent ChatGPT Codex accounts using fresh 5-hour and weekly quota data. It supports explicit synthetic priming for confirmed first-use rolling windows, urgency-based quota draining, replay-safe pre-output failover, and cross-process reservations.
