# Security reporting

Do not report suspected vulnerabilities in a public issue. Use [GitHub's private security advisory form](https://github.com/robinbraemer/pi-quota-router/security/advisories/new) instead.

Do not include OAuth authorization URLs, access or refresh tokens, account ids, or unreviewed diagnostic logs in a report. For the extension's credential, authorization-handoff, and threat boundaries, see [docs/security.md](docs/security.md).
